# MML Hub

## Overview

MML Hub is a secure client portal providing access to work order data and operational tools.

---

## Current Status

- Functional MVP (DEV)
- Running on Azure App Service
- CI/CD enabled via GitHub

---

## Key Features

- Work order viewing and filtering
- Job logging
- Role-based UI behaviour
- Multi-client support (simulated)

---

## Next Steps

- SQL integration
- Authentication (Entra ID)
- Role-based access control

---

## Purpose

To provide a scalable, maintainable portal that complements existing CAFM systems.
